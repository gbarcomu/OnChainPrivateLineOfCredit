## Run locally

1. Clone the repo

2. Install the dependencies

```sh
npm install
```

3. Start the local test node

```sh
npx hardhat node
```

4. Deploy the contracts

```sh
npx hardhat run scripts/deploy.js --network localhost
```

5. Run the app

```sh
npm start
```

6. Execute command line app

```sh
npx ts-node bankCommandLineApp/{fileName}.ts
```
